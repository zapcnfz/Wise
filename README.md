## Wise 介绍

Wise（原名TransferWise）成立于2011年，总部位于英国伦敦，是全球领先的国际汇款和多币种账户服务提供商。截至2024年，Wise已为全球超过1600万用户提供服务，每月处理超过100亿美元的跨境交易。

Wise的核心优势在于透明的汇率机制——采用真实的中间市场汇率（路透社等金融机构使用的实时汇率），而非传统银行的加价汇率，能为用户节省高达8倍的汇款成本。平台支持160多个国家和地区的50多种货币转账，覆盖80多种货币的账户持有功能。

Wise提供的主要服务包括：国际汇款、多币种账户（Wise Account）、实体和虚拟借记卡、商业账户等。特别适合跨境电商从业者、自由职业者、留学生、海外工作人员以及有国际业务往来的中小企业使用。
